My First Project in C

https://github.com/kellerHuang/SomethingAwesomeCS1917

This was a small game in C with ascii art created during my first year of university. The goal for me in this assignment was mainly to just familiarise myself with C and get comfortable with organising longer code with multiple .h and .c files. It was really a blast working on this project and it really did feel good to have others run your code and try break it in every possible way.
