Project 3

These are some small tasks that I've completed using a mix of Python, Perl and Bash

Link To Tasks: 2041 Task Files

These are the weekly tests I did during my 2041 course. These tests were completed in strictly under an hour.

I think that this was honestly the most entertaining part of the COMP2041 course I did at UNSW. If anything, some of these tasks were perhaps a tad too simple for the 1 hour time limit. However, these tasks really show the importance to have a certain "feel" for coding, where the niches of each language could break your program without you even knowing it, and possibly taking a long time to fix. A good example I encountered would be sys.argv including the script name in python, whilst in perl, the 0th member of @ARGV would not be the script name, as well as differences in python's len, as compared to perl's length functions.

These tasks for me felt really useful in appreciating each different language and what would be easy in one language and difficult in another. Coming from compiled languages such as C and Java, these scripting languages are vastly easier to put together for these smaller tasks yet still capable of creating larger projects with useful imported modules such as Requests to make webpages, NumPy to do calculations and wxPython for GUI.
